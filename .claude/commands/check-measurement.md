# 計測フロー確認

現在の計測フローを確認し、問題があれば修正してください:

1. **LocationService** が正しくGPS座標を取得できるか確認
   - 権限リクエストのフローは正しいか
   - タイムアウト処理は適切か

2. **SpeedTestService** の接続確認
   - Cloudflare エンドポイントにアクセスできるか
   - DL/UL/Ping のそれぞれが正しく計測されるか
   - エラー時に null を返すか

3. **NetworkInfoService** の情報取得確認
   - 接続種別 (wifi/cellular) が取得できるか
   - セルラー世代が取得できるか

4. **LoggingService** のCSV出力確認
   - ヘッダー行が正しいか
   - データ行のカラム数がヘッダーと一致するか
   - UTF-8 BOM が付与されているか
   - 特殊文字(カンマ、改行)がエスケープされているか

5. **全体フロー** の動作確認
   - 開始ボタン → 初回計測 → CSV保存 → ポーリング → 停止 の一連の流れ
   - 計測中のUI更新が正しいか
